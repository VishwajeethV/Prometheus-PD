# Prometheus-PD — Website Redesign Scaffold

This repository branch contains a starter Next.js + TypeScript + Tailwind + Framer Motion scaffold for the VH4 Aerospace website redesign. The content and copy originate from the provided investor pitch deck (VH4_Aerospace_PitchDeck.html) and must not be changed outside the deck's facts.

Structure:
- pages/ — Next.js pages
- components/ — React components for each major section (hero, problem, solution, tech, timeline, market, competitive, funding, vision, footer)
- public/ — static assets (placeholders)
- styles/ — global Tailwind + small custom CSS

Next steps: run `npm install` and `npm run dev` to preview.
